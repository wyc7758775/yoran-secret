就如《JavaScript 忍者秘籍》中对于IIFE的描述一样, 函数表达式 + () = 立即函数.

所以有如下:

JavaScript
const a = 1 + function() {return 3} () // 4

w3c 的推荐写法：

JavaScript
(function(){}())

别的常用的写法：

JavaScript
(function(){})()

以及在函数前面加上! && || - + 等符号也可以把函数声明变为函数表达式

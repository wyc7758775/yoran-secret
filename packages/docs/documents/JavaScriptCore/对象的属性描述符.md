**配置definePropertye之后,除了value,其他值不显型的写出的话,默认是false**

```js
var o = { a: 1 };
Object.defineProperty(o, "b", {
	value: 2, 
	writable: false, 
	enumerable: false, 
	configurable: true,
});
```
- value: 基本属性值
- writable: 是否可以改,如果为false的话,在es5严格模式下面就会报错了
- enumberable: 是否可以枚举
	设置为false的话,for in就出不来这个属性
- configurable: 是否可以配置

	```js
	const obj = {
		value: 6
	}	
	Object.defineProperty(obj, 'value', {
		value: 2, 
		writable: true, 
		enumerable: true, 
		configurable: false,
	})


	Object.defineProperty(obj, 'value', {
		value: 4, 
		writable: true, 
		enumerable: true, 
		configurable: true,
	})
	```
	第二次defineProperty直接报错!!!!
